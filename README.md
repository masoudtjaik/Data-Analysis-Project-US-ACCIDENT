# US Accidents Data Analysis

## Project Overview

This project explores the US Accidents dataset to identify traffic accident patterns and investigate the factors associated with accident occurrence and severity. The analysis includes data cleaning, exploratory data analysis (EDA), and visualizations using Python.

---

## Dataset

- Dataset: US Accidents Dataset
- Source: Kaggle
- Records: Over 7 million traffic accident records
- Features include:
  - Date and Time
  - Location (State, City)
  - Severity
  - Weather Conditions
  - Temperature
  - Humidity
  - Visibility
  - Wind Speed
  - Road Features

---

## Objectives

The project answers the following questions:

- Which states and cities have the highest number of accidents?
- When do traffic accidents occur most frequently?
- How is accident severity distributed across the dataset?
- How do weather conditions influence accident occurrence and severity?
- Are accidents more frequent during the day or at night?
- What are the key factors associated with severe traffic accidents?

---

## Data Cleaning

The following preprocessing steps were performed:

- Removed unnecessary columns
- Handled missing values
- Converted date columns to datetime format
- Extracted Year and Month from the accident date
- Checked duplicate records
- Prepared the dataset for analysis

## Exploratory Data Analysis (EDA)

The project includes visualizations for:

## Key Findings

- California recorded the highest number of accidents.
- Miami had the largest number of reported accidents among all cities.
- Accident frequency peaked during December.
- Most accidents occurred during the daytime.
- Severity Level 2 accounted for the majority of reported accidents.
- Fair weather recorded the highest number of accidents due to higher traffic volume.
- Severe accidents were generally associated with lower temperatures, higher humidity, and slightly higher wind speeds.
